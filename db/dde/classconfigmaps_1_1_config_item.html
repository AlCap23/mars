Detailed Description

Base object of the configmaps library!

This is the basic object that manages elements in the configmaps data structure. It can contain all ConfigBase items, generate, serialize and de-serialize them. Possible items are: ConfigAtom (scalar values), ConfigVectors (lists), and ConfigMaps (key, Item).
