37 /* We had to jump through some hoops to get the git version information
38  * into the libInterface:
39  * 1) If we set it directly in the constructor the linker seems to mess things
40  * up (has only a single code section of the ctor?).
41  * 2) So we set the info in the create_c.
42  * 3) create_c must therefore be a friend function of the LibInterface.
43  * 4) create_c must therefore be forward declared.
44  * 5) The LibInterface and LibManager must therefore be forward declared.
45  * NOTE: because of step 2) the git info only becomes available after the
46  * classes constructor!
47  */
